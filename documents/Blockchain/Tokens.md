# 블록체인 토큰 소개

토큰을 이해하는 것은 Web3와 블록체인 산업을 탐색하는 데 필수적입니다. 토큰은 생태계의 핵심 구성요소로, 탈중앙화 금융(DeFi)부터 디지털 아트까지 모든 것의 기반이 됩니다. NFT(대체 불가능 토큰)와 ERC20 토큰 같은 예시를 들어보셨을 텐데, 이들은 광범위하고 강력한 디지털 자산 카테고리의 일부에 불과합니다.

## 토큰이란?

토큰은 **블록체인 위에 존재하는 가치 또는 유틸리티의 디지털 표현**입니다. 이러한 프로그래밍 가능한 자산은 스마트 컨트랙트에 의해 관리되며, 다양한 개념을 나타낼 수 있습니다:

| 유형       | 설명                                |
| ---------- | ----------------------------------- |
| **통화**   | 네트워크 내 교환 수단               |
| **소유권** | 디지털 또는 물리적 자산의 소유 증명 |
| **투표권** | 프로토콜 거버넌스 참여 권한         |
| **접근권** | 특정 서비스나 기능 해제 키          |

대부분의 토큰은 이더리움의 **ERC(Ethereum Request for Comments)** 같은 특정 기술 표준을 따릅니다. 이러한 표준은 토큰이 예측 가능하게 동작하도록 보장하여, 다양한 애플리케이션, 지갑, 거래소와 원활하게 상호작용할 수 있게 합니다.

---

## 블록체인 토큰의 핵심 유형

### 1. 네이티브 토큰 (Native Tokens)

네이티브 토큰은 특정 블록체인 네트워크의 **기본 통화**입니다. 다른 토큰과 달리 스마트 컨트랙트로 생성되지 않고, 블록체인의 핵심 프로토콜에 내장된 필수 요소입니다.

#### 주요 특징 및 용도

- **가스비 지불**: 트랜잭션 처리와 스마트 컨트랙트 실행에 필요한 연산 작업의 보상
- **프로토콜 상호작용**: 네이티브 생태계 내 결제, 거래 등 온체인 활동의 기본 통화
- **네트워크 보안**: PoS 블록체인에서 네이티브 토큰을 스테이킹하여 네트워크 보안에 기여하고 보상 획득
- **거버넌스**: 일부 네트워크에서 프로토콜 업그레이드 등의 제안에 대한 투표권 부여

#### 예시

| 토큰    | 네트워크                                   |
| ------- | ------------------------------------------ |
| **ETH** | 이더리움 및 L2 솔루션 (Optimism, Arbitrum) |
| **BTC** | 비트코인                                   |
| **SOL** | 솔라나                                     |

---

### 2. 대체 가능 토큰 (Fungible Tokens)

대체 가능 토큰은 **상호 교환 가능하고 분할 가능**합니다. 한 단위의 토큰은 동일한 토큰의 다른 단위와 가치와 기능이 동일합니다. 마치 1달러가 다른 1달러와 같은 것처럼요.

> 이더리움에서 대부분의 대체 가능 토큰은 **ERC20** 표준으로 구축됩니다.

#### 대체 가능 토큰의 종류

| 종류              | 설명                                               | 예시             |
| ----------------- | -------------------------------------------------- | ---------------- |
| **스테이블코인**  | 외부 자산(주로 USD)에 페깅된 안정적 가치 유지 토큰 | USDC             |
| **유틸리티 토큰** | 프로토콜 내 특정 제품/서비스 접근권 부여           | LINK (Chainlink) |
| **거버넌스 토큰** | DAO에서 투표권 부여                                | UNI (Uniswap)    |
| **보상 토큰**     | 프로토콜 참여(유동성 제공 등)에 대한 보상          | -                |

---

### 3. 대체 불가능 토큰 (NFT)

NFT는 **상호 교환 불가능한 고유 디지털 자산**입니다. 각 NFT는 고유한 아이덴티티와 속성을 가지며, 세상에 하나뿐인 토큰입니다.

#### 주요 특징

- ✅ **고유성**: 모든 NFT는 검증 가능하게 다름
- ✅ **비교환성**: 1:1 기준으로 다른 NFT와 교환 불가
- ✅ **비분할성**: 더 작은 단위로 분할 불가
- ✅ **메타데이터**: 이름, 설명, 이미지/파일 등 속성 정보 연결

> 이더리움에서 NFT의 가장 일반적인 표준은 **ERC721**입니다.

#### 활용 사례

- **디지털 아트**: CryptoPunks, Bored Ape Yacht Club 등 디지털 창작물 소유권 증명
- **ENS 도메인**: 이더리움 지갑 주소의 사람이 읽을 수 있는 이름 (예: yourname.eth)
- **게임 내 자산**: 블록체인 기반 게임의 고유 아이템, 캐릭터, 가상 토지

---

### 4. 반대체 가능 토큰 (Semi-Fungible Tokens)

반대체 가능 토큰은 **대체 가능과 대체 불가능의 하이브리드**입니다. 동일한 복사본이 여러 개 존재할 수 있지만, 각 복사본은 개별적으로 추적되고 소유됩니다.

> 가장 일반적인 기술 표준은 **ERC1155**입니다.

#### 활용 사례

| 사례               | 설명                                                                          |
| ------------------ | ----------------------------------------------------------------------------- |
| **이벤트 티켓**    | 500장의 "일반 입장권"은 서로 대체 가능하지만, 50장의 "VIP 티켓"과는 대체 불가 |
| **게임 내 아이템** | 수천 개의 "체력 포션"은 기능상 동일하지만 각각 고유 토큰으로 거래 가능        |
| **토큰 게이팅**    | 보유한 토큰 클래스에 따른 서로 다른 등급의 콘텐츠 접근                        |
| **자격증**         | 여러 개인에게 발급되는 학위나 전문 자격증                                     |

## 실물자산 토큰화 (RWA)

Web3에서 빠르게 성장하는 영역으로, **물리적 또는 전통 금융 자산의 소유권을 나타내는 디지털 토큰**을 블록체인에 생성하는 것입니다. 토큰화는 전통 금융(TradFi)과 DeFi 사이의 간극을 메우는 것을 목표로 합니다.

### 토큰화되는 자산 예시

- 🏠 부동산
- 🏛️ 미국 국채
- 🥇 금
- 💼 사모펀드

### 토큰화의 장점

| 장점            | 설명                                                                      |
| --------------- | ------------------------------------------------------------------------- |
| **접근성 향상** | 분할 소유를 통해 부동산, 미술품 등 비유동 자산에 더 많은 투자자 참여 가능 |
| **거래성 개선** | 전통 시장 시간에 제한받지 않고 24/7 글로벌 거래 가능                      |
| **효율성 증대** | 중개인과 수작업 서류 의존도 감소로 더 빠르고, 저렴하고, 투명한 거래       |

## 스테이블코인 심층 분석

스테이블코인은 DeFi 생태계의 초석으로, 다른 암호화폐의 변동성 없이 **안정적인 교환 수단과 가치 저장 수단**을 제공합니다.

### 스테이블코인의 3가지 유형

| 유형              | 설명                                                    | 예시       | 특징                                                     |
| ----------------- | ------------------------------------------------------- | ---------- | -------------------------------------------------------- |
| **법정화폐 담보** | 예비 은행 계좌에 1:1로 보유된 법정화폐로 담보           | USDC, USDT | 매우 안정적이지만 중앙화됨 (법적 요구 시 자산 동결 가능) |
| **암호화폐 담보** | 스마트 컨트랙트에 잠긴 다른 암호화폐로 담보             | DAI        | 탈중앙화, 변동성 방지를 위해 초과 담보                   |
| **알고리즘 기반** | 복잡한 알고리즘과 시장 메커니즘으로 토큰 공급 자동 관리 | -          | 가격 페그 유지                                           |

## 토큰 획득 및 거래 방법

### 중앙화 거래소 (CEX)

- **플랫폼**: Coinbase, Binance 등
- **특징**: 신뢰할 수 있는 중개자 역할, 사용자 자금 수탁 관리
- **거래 방식**: 전통적인 오더북 시스템

### 탈중앙화 거래소 (DEX)

- **플랫폼**: Uniswap, Curve 등
- **특징**: 자체 수탁 지갑에서 직접 P2P 거래
- **거래 방식**: 스마트 컨트랙트를 통한 투명한 온체인 실행

## 토큰 표준 요약

| 표준        | 유형              | 특징                                |
| ----------- | ----------------- | ----------------------------------- |
| **ERC20**   | 대체 가능         | 상호 교환 가능, 분할 가능           |
| **ERC721**  | 대체 불가능 (NFT) | 고유, 비분할                        |
| **ERC1155** | 반대체 가능       | 하이브리드, 효율적인 다중 토큰 관리 |
